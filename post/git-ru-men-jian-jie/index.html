Git入门简介

2019-08-10

Git简单使用


git init            //初始化本地仓库
git add .           //添加文件到版本库（只是添加到缓存区），.代表添加文件夹下的所有文件
git commit -m "first commit"             //添加要提交文件的描述信息
git remote add origin +你的远程仓库地址     //链接到远程仓库，创建主分支
git push -u origin master                //将本地的master分支推送到origin主机的master分支。如果后者不存在，则会被新建
git pull origin master   //取回远程仓库主机或某个分支（或master）的更新，再与本地的指定分支合并。

Git名词解释

拉取、获取、合并、提交、推送、同步

  1. 拉取（Pull）：将远程版本库合并到本地版本库；相当于（Fetch + Meger）
  2. 获取（Fetch）：从远程仓库获取最新版本
  3. 合并（Meger）：将两个版本库进行合并操作
  4. 提交（Commit）：将所做的更改，存入本地暂存库
  5. 推送（Push）：将所做的更改，存入远程版本库
  6. 同步（Pull + Push）：先将远程库拉取，再将本地库推送到远程库（注意顺序！）；相当于（Pull+ Push）
